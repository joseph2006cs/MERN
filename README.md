# Task Manager

A lightweight task management app built with vanilla JavaScript. Add, complete, and delete tasks with persistent local storage.

## Features

- Add and delete tasks
- Mark tasks as complete/incomplete
- Tasks automatically saved to browser storage
- Dark theme UI
- Mobile responsive

## How to Use

1. Open `index.html` in your browser
2. Type a task and click "Add Task" or press Enter
3. Click the checkbox to mark tasks complete
4. Click the trash icon to delete tasks

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Local Storage API
